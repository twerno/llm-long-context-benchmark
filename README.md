# Node.js + TypeScript Project

Czysty projekt Node.js z TypeScript.

## Instalacja

```bash
npm install
```

## Uruchomienie

### W trybie deweloperskim (z ts-node):

```bash
npm run dev
```

### Budowanie:

```bash
npm run build
```

### Uruchomienie po budowaniu:

```bash
npm start
```

## Struktura projektu

```
.
├── src/
│   └── index.ts      # Główny plik źródłowy
├── dist/             # Wyjście kompilacji (utworzone po build)
├── package.json
├── tsconfig.json
└── README.md
```
